**Status:** Now matches Rockwell et al. (2021) Table 4 **EXACTLY**

**What was wrong:**
- Had 21 classes with wrong logic (Classes 15, 16 didn't exist in paper)
- Priority order was incorrect
- Class conditions didn't match paper specifications

**What's fixed:**
- **19 paper classes** with exact Boolean logic from Table 4

- **Correct priority order:** Low priority first, iron sulfate classes override, water classes highest priority

**Example fixes:**
```javascript
// OLD (WRONG):

```

### 2. Dark Mask Threshold - FIXED ✅
**Status:** Corrected to paper specification

**What was wrong:**
```javascript
darkMaskThreshold: 0.12  // WRONG!
```

**What's fixed:**
```javascript
darkMaskThreshold: 0.2125  // Paper: B6 DN < 15000 = 15000*0.0000275-0.2 = 0.2125
```

**Impact:** Properly filters dark shadows and rock areas as per paper methodology.

---

### 3. Sentinel-2 Band Resampling - FIXED ✅
**Status:** All bands now properly resampled to 30m

**What was wrong:**
- B1 (60m) used directly without resampling
- B2-B4, B8 (10m) not aggregated to 30m
- B11-B12 (20m) not resampled to 30m
- Mixed resolutions caused spatial inconsistencies

**What's fixed:**
```javascript
// B1: 60m → 30m (bicubic interpolation)
var b1_30m = image.select('B1').resample('bicubic');

// B2, B3, B4, B8: 10m → 30m (mean aggregation)
var b10m_30m = image.select(['B2', 'B3', 'B4', 'B8'])
  .reduceResolution({reducer: ee.Reducer.mean(), maxPixels: 1024});

// B11, B12: 20m → 30m (bicubic)
var b20m_30m = image.select(['B11', 'B12']).resample('bicubic');
```

**Impact:** Sentinel-2 now matches Landsat 30m resolution for consistent multi-sensor analysis.

### 4. Turbidity Index - ADDED ✅
**Status:** New index for contaminated water quality assessment

**What's new:**
```javascript
// Turbidity Index: B4/B2 (red/blue ratio)
// Higher values = more suspended sediments (AMD precipitation)
var turbidity = b4.divide(b2_safe).rename('Turbidity');
```

**Used in:**
- Contaminated water detection (turbidity > 1.3)
- Click handler output
- Water quality assessment

---

### 5. Enhanced Water Detection - IMPLEMENTED ✅
**Status:** Your custom method with multi-index approach

**Features:**
- **MNDWI** (primary): `(B3-B6)/(B3+B6) > 0.30`
- **AWEInsh** (secondary): `B2 + 2.5×B3 - 1.5×B5 - 0.25×B7 > 0.0`
- **NDWI** (optional): `(B3-B5)/(B3+B5) > threshold` (not default)
- **Turbidity check**: `B4/B2 > 1.3` for contaminated water

**Contaminated water logic:**
```javascript
var isContaminatedWater = iron.gt(1.8)
  .and(isWater)
  .and(turbidity.gt(1.3));
```

### 6. Classification Priority Order - FIXED ✅
**Status:** Matches paper hierarchy

**Correct order:**
1. **Classes 1-11:** Non-iron sulfate (lowest priority)
2. **Classes 12-19:** Iron sulfate minerals (high priority - override above)
3. **Classes 20-21:** Water (HIGHEST priority - override everything)

**Critical classes (highest to lowest NAP):**
- **Class 14:** Oxidizing Sulfides (NAP=1) - MOST CRITICAL
- **Class 12:** Major Iron Sulfate (NAP=2)
- **Class 17:** Proximal Jarosite (NAP=3)
- **Class 18:** Distal Jarosite (NAP=4)
- **Class 9:** Argillic Alteration (NAP=5)
- **Class 19:** Clay + Ferrous + Iron (NAP=6)
